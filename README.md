## How to Install

Make sure mysql already started.
```bash
git clone git@github.com:agilejzl/blog-eye.git
cd blog-eye
# A ruby script to RTF
ruby setup.rb
rails s
```

## How to Test

We use rspec to test.
```bash
bundle exec rake rspec
```
